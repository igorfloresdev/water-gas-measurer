# Commands

### To install just:
<br>
<pre>npm install</pre>

### To build:
<br>
<pre>npm run build</pre>

### To start:
<br>
<pre>npm run start</pre>

### To dev:
<br>
<pre>npm run dev</pre>

<br>

# Docker Commands

<pre>docker compose build --no-cache</pre>
<pre>docker compose up -d</pre>
